# Commands

## Template safety (once per new repo clone)
- Enable template guard hook: `git config core.hooksPath .githooks`
- Bypass once (if needed): `SKIP_TEMPLATE_GUARD=1 git commit -m "..."`.

## Install
- (if needed) e.g. `npm install`

## Run
- e.g. `npm run dev`

## Test
- e.g. `npm test`

## Lint/Format
- e.g. `npm run lint`
